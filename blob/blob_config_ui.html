🔧 Blob Storage Configuration

Frontend-only configuration using SAS tokens (no backend needed!)

How to generate SAS token:
1. Go to Azure Portal → Storage Accounts → Your Account
2. Under "Security + networking" → "Shared access signature"
3. Set permissions: Read, Write, Delete, List
4. Set expiry date (e.g., 1 year from now)
5. Click "Generate SAS and connection string"
6. Copy the SAS token (starts with ?sv=...)
7. Make sure container exists (e.g., "marketing-assets")
Your Azure Storage account name (no .blob.core.windows.net)
Shared Access Signature token (starts with ?sv=... or sv=...)
The container must already exist in your storage account
Folder path within the container (leave trailing slash)
